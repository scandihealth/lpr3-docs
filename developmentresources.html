Development resources

Schematron

Schematron definitions used to validate requests to the IHE/XDR HL7 CDA interface of the Danish National Patient Register are available as is for integrators. The entry point Schematron definition is called lpr-lpr3fr.sch.

SOAP UI project

A SOAP UI project that calls an endpoint in the test environment is available as is for integrators.
